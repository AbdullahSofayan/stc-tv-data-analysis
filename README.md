## STC Virtual Work Experience – Data Analysis

This course is a virtual work experience program provided by **STC**, designed to simulate real-world tasks performed by data analysts and data scientists in the telecommunications industry.

During this experience, participants work with real datasets and learn how to analyze data, build predictive models, and present insights using data storytelling techniques. The program is suitable for **beginner and intermediate learners** who are interested in developing practical skills in data analysis.

### Learning Outcomes
After completing this virtual experience, participants are able to:

- Identify and use appropriate **statistical analysis tools** (Task 1)
- Build **prediction models** to forecast user behavior (Task 2)
- Develop **machine learning algorithms** for recommendation systems (Task 3)

### Skills Gained
The following skills are developed during this program:

- Statistical analysis tools  
- Predictive data modeling  
- Machine learning for recommendations  
- Data-driven decision making  
- Data storytelling and presentation

This experience provides hands-on exposure to how data is used within organizations like STC to support decision-making and improve user experience.
